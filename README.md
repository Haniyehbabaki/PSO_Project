# Particle Swarm Optimization (PSO) Implementation

This project is part of the **Computational Intelligence course**.  
The goal is to implement the **Particle Swarm Optimization (PSO)** algorithm to find the **minimum of the function**:

## üìò Description

This project simulates a simple **PSO algorithm** that optimizes a two-dimensional function.  
Each particle explores the search space, updating its position and velocity based on:
- Its **personal best position**
- The **global best position** of the swarm
- The **inertia weight** and **acceleration coefficients**
